# Project Overview

## Purpose

This project demonstrates a practical C# console application using object-oriented programming and LINQ. It is designed as a portfolio-ready project for beginner-to-intermediate .NET developers.

## Architecture

The project separates responsibilities into four main layers:

- `Models`: Entity classes such as `LibraryItem`, `Book`, `Magazine`, `Dvd`, and `Member`.
- `Services`: Business logic in `LibraryService`.
- `Data`: Seed data for demo usage.
- `Utilities`: Console input, output, and validation helpers.

## Added Features Beyond Basic CRUD

- Borrow and return flow
- Due dates
- Overdue detection
- Member records
- Most borrowed report
- Inventory reports by type and category
- Input validation
- Color-coded console output
